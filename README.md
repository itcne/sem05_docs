# Semesterarbeit 05 - Backstage.io – Simplifying Unified Onboarding for Kubernetes

[![GitHub Pages - Docs](https://github.com/Cloud-native-engineering/sem04_docs/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/Cloud-native-engineering/sem04_docs/actions/workflows/pages/pages-build-deployment)

In der heutigen agilen und VUCA-Welt (Volatility, Uncertainty, Complexity, Ambiguity) verändert sich die Arbeitsumgebung stetig. Dies führt zu fortlaufenden Anpassungen in Prozessen und Technologien, insbesondere bei der Integration von Anwendungen in moderne Infrastrukturen.

Backstage, ursprünglich bei Spotify entwickelt, ist eine Open-Source-Software, die als zentrales Entwicklerportal dient. Als Entwicklerportal vereinfacht und standardisiert Backstage den Onboarding-Prozess, indem es alle relevanten Tools in einer einheitlichen, benutzerfreundlichen Oberfläche bereitstellt. Als Incubating-Projekt der Cloud Native Computing Foundation (CNCF) wird Backstage aktiv unterstützt und weiterentwickelt.

Diese Arbeit untersucht, wie Backstage genutzt werden kann, um Entwicklern den Einstieg in Kubernetes zu erleichtern und den Integrationsaufwand zu minimieren.
